# 研究西方政治制度史的意义
对外交往愈加频繁，为更好认识我国与西方政治制度提供依据，加快我国政治体制改革。

| 制度           | 国家                                                 |
| -------------- | ---------------------------------------------------- |
| 奴隶主国家制度 | 古希腊、古罗马                                       |
| 封建制         | 法兰西、意大利、德意志、英吉利、东西比较             |
| 资本主义制度   | 英国(君主立宪)、美国(三权分立)、法国(半总统制)、德国 |
