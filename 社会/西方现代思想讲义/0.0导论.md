# 01、思想的现实意义
## 1、思想内在于现实
**思想观念是内在于社会现实的，观念是驱动实践行为的构成要素**。
>把思想和现实对⽴起来，看成两种分离的东⻄，虽然很流⾏，却 是完全错误的。你可能会说，我知道“思想离不开现实”，但我要强调 的观点⽐这还要深⼊⼀步，是说“现实离不开思想”。如果离开了思 想，根本不存在“社会”现实，当然也谈不上去理解现实。
### 举例：购物
#经济人
购买性价比最优的商品，背后存在着“性价比最优”的思想逻辑，即经济学的基本假设——“经济理性人”模型。
- 人是自利的，首要关切追求自己的利益
- 人是理性的，会理性计算成本与收益
## 2、利益并不是物质性的
#符号消费
名牌消费的现象不能简单地从物质利益角度来理解，而必须放在一个文化意义的结构中来把握。实际上的价值，必须依据一个思想观念的框架。

## 3、人类行为的复杂动机结构
思想观念是⼈的社会实践⾏动的驱动要素，这在经济领域之外表 现得更为复杂多样。⽐如，有⼈为爱情可以放弃许多物质利益，还有 ⼈为某种理想或信仰可以牺牲很多世俗的利益。这是简单的“⾃ 利”和“理性”概念⽆法解释的。
### 举例：春运
如果对大规模的春运现象做一种“经济理性”利益分析，就会发现“回家过年”无非是与家人团聚、与亲朋见面，吃饭、娱乐。从理性计算的角度看性价比极低。如离开了春节对中国人的文化意义，以及人们的孝敬与感恩等思想观念，将无从解释春运现象。

**思想观念内在于社会现实，是社会实践行动的驱动要素**。
# 02、现代与现代性
#时间意识 #现代性 
## 1、现代是一种新的时间意识
### 字面含义：当下或此刻
即当下的时代，16C被创造，也就是说16C之前的西方，人们不太关心“当下的时代”，缺乏对当下的敏感度
### 思想观念层面：对当下新颖性的敏感
一种新的时间意识，或者历史观
#### 循环历史观
#循环历史观
古人观念之中，当下的观念不过是以往时代的延续和重复。人们感受到的时间是不断循环的，来源于自然观念的启发，对应着循环的时间意识。
#### 新的时间观念
文艺复兴，特别是启蒙运动和工业革命之后，人们对“当下间成为了一个有方向的矢量观念
- 传统不再可靠。生活不是循环往复，那么过去的经验很可能不再可靠
- 时间观念的转变。由“厚古薄今”转向“厚今薄 古“。形成了”线性进步的历史观“
- 创造性和主体性的肯定。人从循环历史宿命的束缚中解放的时代”表现出了越来越强的敏感。当下的时代不再是以往的延续和重复，时，成为自由的、有目的的创造者
>《共产党宣⾔》中写过这样 ⼀句话，“⼀切固定的东⻄都烟消云散了，⼀切神圣的东⻄都被亵渎 了”。这句话⽣动地表达了“现代”的含义：现代就意味着崭新的重⼤变 ⾰，也可以被称作“古今之变”。
## 2、现代化的过程与现代性的结果
### 一、引发现代变革的动力
激发现代变革的主要思想动力是理性的观念，确切地说是“启蒙理性主义"。
**社会现实的因素与思想观念的因素相互作用、彼此强化导致了现代的转变**
### 二、现代化的结果
西方形成了现代社会。经济上，现代工商业、城市崛起；政治上，民族国家形成，现代民主与宪政发展；社会层面，人口大规模流动(地域与阶层)；思想文化方面，理性主义获得主导地位，自由、平等和个人权利意识兴盛。

## 动力来源
"理性"的观念
>古希腊的理性主义传统经过漫⻓⽽复杂的历史传承，在18世纪的 启蒙运动中达到了巅峰，发展为现代的启蒙理性主义。当时，德国的 ⼤哲学家康德说过，启蒙运动的⼝号就是⼤胆地“运⽤你⾃⼰的理 智”！只有这样⼈类才能摆脱蒙昧的“不成熟状态”。他还主张，启蒙时 代是⼀个批判的时代，就是⽤理性来检查以往未经反思的信念，⽆论 是神圣的上帝，还是威严的法律，都要接受理性的检验。他说：“只有 经得起理性的⾃由、公开检查的东⻄才能博得理性的尊敬”。
>
因此，启蒙就是⽤理性的光芒打破⿊暗，让⼈摆脱⾮理性的蒙 昧，⾛向成熟。理性成为区分真理与蒙昧的决定性标准，成了衡量⼀ 切的准绳；理性甚⾄取代了神的位置，具有近乎上帝⼀般的神圣地 位。这是启蒙理性主义的确切含义。在思想层⾯上可以说，启蒙理性 主义是⻄⽅古今之变的核⼼
## 思想转变
- 人类中心主义的转变
	- 人从自然中脱离出来(*古希腊的时候，有认识自然但不提倡改造自然。因为自然是活生生的有机整体，改造自然将使自然失去生命*)。现代人在观念上，成为了”考察者“，人身处在自然中，但在科学主导的思想观念中与自然脱离，站在了自然世界的对立面。且人处于积极主动的主体地位，自然处在消极被动的客体地位。
	- 这一转向促使人类开始探索、改造、利用自然，带来了工业化和城市化的迅速扩张，与深重的生态问题
- 个人主义的转变
	- 传统社会中，个人深深嵌入于群体之中。但在现代社会，尽管个⼈依然离不开社群，但总是可以离开任何⼀个特定的社群。这带来了难以想象的个人自由，生活变得多样而丰富；代价则是失去了与⼀个特定群体的久远、厚重和牢固的纽带关系，这带 来了孤独感、漂泊感和乡愁。这也是现代性困境的⼀部分。
