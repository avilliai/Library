# 党政体制的基本特征——一元化领导

党政不是平行关系，而是主次关系  
含义：党组织对一切国家机构实行组织关系上的

一元化领导的主要特征：其一，权力中心的唯一性；其二权力结构的集中性；其三，权力形态的统合性
（以党建政、政党领导、政党集权、政党统合）
# 因为中国是一个高度行政化的国家，所以很多单位也有职级
